# ddl-annotation-processor

Intro
---

This Annocation proccesor can create a query - in a form of a file- which generates mysql database table.

The file will be created at the root path in the project that you ran.

Support Annotation
--- 

- Entity 
- Column
- Embedded
- AttributeOverride
- Inheritance
- Comment

# ddl-annotation-processor

Intro
---

This Annotation Proccesor can create a query for mysql database table in the form of a file.

The file will be created at the root path in the project that you ran.

Support Annotation
--- 

- Entity 
- Column
- Embedded
- AttributeOverride
- Inheritance
- Comment
